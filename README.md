# ReferenceArtisan

Crear una página en la que se guardaría y se iría anotando todos los proyectos que se hiban haciendo.

 - __Nombre del proyecto.__ Cosas como, __Aprendiendo PHP__ y también con sus subtemas __Creación de variables__, entre otros.
 - __Nombre del tema.__ Aquí se especificaba el tema que estaba contextualizado. Por ejemplo el anterior tema era __Aprendiendo PHP__ su tema sería __Programación__.
